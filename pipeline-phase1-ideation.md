# Phase 1: Ideation Results

## Agent 1 (Grok-ideator) Ideas

### 1. Gravity Painter
Draw luminous trails with particles that fall, bounce, and swirl through gravity wells you place on the canvas.

Combines the meditative flow of drawing with emergent physics behavior—every stroke becomes a living system. The interplay between your intentional marks and physics creates unexpected beauty.

Technical: Particle system with configurable gravity wells, Verlet integration, Canvas API/WebGL, Quadtree spatial partitioning. Vue 3 reactive state for gravity well positions.

Expansion: Preset modes, particle types (fire, water, light, magnetized), advanced wells, gallery mode, challenge mode.

### 2. Chromatic Cipher
Decode color patterns using logic rules where each hue represents a constraint in an evolving puzzle grid.

Turns abstract color theory into tangible logic puzzles—like Sudoku meets synesthesia. Players learn to "read" color relationships spatially.

Technical: Constraint satisfaction solver, graph coloring algorithms, reactive grid state with Vue 3 computed properties, SVG/Canvas, procedural generation.

Expansion: Rule variants, hex grids, timed modes, color blindness modes, daily challenge.

### 3. Cell Symphony
A musical cellular automaton where living cells trigger notes, creating evolving generative compositions.

Bridges Conway's Game of Life with music composition, revealing how complex melodies emerge from simple rules.

Technical: Modified Game of Life engine, cell properties map to musical parameters, Web Audio API, sparse grid optimization. Vue reactive grid triggers CSS animations.

Expansion: Rule presets, scale modes, instrument banks, interactive seeding, performance recording.

### 4. Orbital Garden
Design solar systems where planets trace art through their orbital paths, creating spirograph-like mandalas.

Merges orbital mechanics with generative art—every planetary arrangement produces unique geometric patterns. Educational and artistic.

Technical: N-body gravity simulation, orbital path tracing, Canvas API, parametric equations for orbital art. Vue reactivity for real-time parameter adjustments.

Expansion: Multi-star systems, moons/asteroids, orbital resonances, export as SVG/PNG, speed controls, collision events.

### 5. Tessellation Studio
Create and explore tessellating patterns through interactive deformation of geometric tiles.

Makes the mathematics of tessellation tangible and artistic. Users discover how shapes fit together through direct manipulation.

Technical: Computational geometry for tessellation validation, SVG path morphing, Vue reactivity for tile parameter binding to CSS transforms, WebGL for complex tilings.

Expansion: Wallpaper group presets, Escher-style metamorphosis, 3D tessellations, animation modes, export to fabric/papercraft patterns.

## Agent 2 (Kimi-ideator) Ideas

### 1. Chromatic Drift
Interactive color space explorer through RGB, HSL, and LAB color spaces with fluid reactive gradients.

Makes color relationships tangible and playful. Vue 3's reactivity shines: cursor position directly drives CSS custom properties.

Technical: Vue 3 reactive state mapping coordinates to color parameters, CSS custom properties, color conversion algorithms, gradient mesh generation.

Expansion: Color space modes, famous palette presets, color matching challenges, export as CSS/design tokens, WCAG contrast ratios.

### 2. Flock Mind
Boids simulation where thousands of particles exhibit emergent flocking behavior.

Emergence is magical—simple rules create complex, lifelike behavior. Users become part of the ecosystem.

Technical: Canvas-based rendering, Boids algorithm (separation, alignment, cohesion), spatial hashing/quadtree, force vectors from mouse interaction.

Expansion: Interaction modes (predator, attractor, wind, obstacle), flock types (fish, birds, insects), obstacle drawing, behavior toggles, stats overlay.

### 3. Prime Spiral
Visualize prime number patterns through interactive Ulam spirals and Sacks spirals.

Transforms abstract number theory into stunning visual art. Diagonal rays and curves emerge from "random" primes.

Technical: Prime sieve, Canvas rendering with coordinate mapping, zoom/pan controls, Web Workers for computation, Vue reactivity for display options.

Expansion: Different spiral types, zoom into massive primes, color by prime gaps/twins, export high-res, educational tooltips, unsolved conjectures overlay.

### 4. Orbital Weave
Physics toy where you place gravitational bodies and watch particles weave trajectories through the system.

Combines the satisfaction of trajectory prediction with the beauty of generative line art.

Technical: N-body physics simulation, trajectory prediction with RK4 integration, particle trail rendering, Vue reactivity for parameter UI.

Expansion: Black holes, binary stars, Lagrange point challenges, artistic filters on trails, trajectory prediction mode, "paint with orbits" challenges.

### 5. Memory Weave
Spatial memory puzzle where patterns must be recalled across a woven, multi-dimensional grid.

Tests working memory and spatial reasoning in a unique format. The woven structure adds complexity.

Technical: Multi-dimensional array representation, pattern generation algorithms, Vue reactivity for grid state, CSS Grid/Flexbox for woven layout, progressive difficulty.

Expansion: Grid dimension increases, interference patterns, time pressure modes, daily puzzles, competitive multiplayer.

## Agent 3 (Gemini-ideator) Ideas

### 1. Particle Calligraphy
Draw with intelligent ink particles that flow, split, and recombine based on velocity and attraction fields.

Your strokes remain alive—particles continue to interact after you've drawn them.

Technical: Canvas-based particle system with quadtree, Verlet integration, attraction/repulsion fields, Vue reactive properties bound to particle stats.

Expansion: Brush presets, interactive challenges, export modes, collaborative canvas, physics parameters control.

### 2. Modal Maze
Navigate a shifting labyrinth by playing musical modes to open paths.

Teaches music theory through spatial puzzle-solving. Each mode has distinct emotional character.

Technical: Web Audio API, algorithmic maze generation, pitch detection, Vue reactivity for maze wall transformations.

Expansion: Difficulty progression, time pressure, chord extensions, adaptive difficulty, multiplayer relay.

### 3. Decay Garden
Cultivate forms that slowly deteriorate through entropy and organic decay patterns.

Explores wabi-sabi aesthetics digitally. Vue's reactivity binds CSS variables to decay state.

Technical: Cellular automata with decay rules, Perlin noise for erosion, SVG morphing, Vue computed properties → CSS filters/transforms.

Expansion: Plantable forms, environmental factors, decay reversals (preservation tools), decay galleries, philosophical prompts.

### 4. Tesseract Fold
Manipulate 4D hypercubes projected into 3D space, solving puzzles by folding through the fourth dimension.

Makes higher-dimensional geometry intuitive through interaction. Mind-expanding spatial reasoning.

Technical: 4D matrix transformations, projection matrices (4D→3D→2D), Vue reactivity for rotation controls, CSS 3D transforms, educational annotations.

Expansion: Different 4D shapes, puzzle mode, stereographic projection modes, guided tutorials, VR preparation mode.

### 5. Breath Geometry
Microphone-driven generative art where breathing patterns create crystalline geometric forms.

Combines mindfulness with creativity. Creates personal mandalas from breath rhythm.

Technical: Web Audio API for microphone input, breath pattern detection, reactive particle systems driven by breath data, Vue reactivity for real-time shape generation.

Expansion: Breath coaching modes, different geometric systems, session tracking, sharing breath mandalas, meditation timer integration.
