## 📝 **Prompt Template:**

```bash
mdls "[Action word] [Topic] [Constraint/Format] [Audience/Analogy]"

# Examples:
# Action: Explain, Design, Write, Compare, Create
# Topic: quantum computing, morning routine, alien creature
# Constraint: in 100 words, using food metaphors, as a sonnet
# Audience: to a 10-year-old, to your grandmother, for beginners
```

## 🔧 **Customize Your Own:**

```bash
# Pick one from each column:
# Action: Explain | Design | Write | Compare | Create
# Topic: [your interest here]
# Constraint: in exactly [N] words | using only [X] metaphors | as a [format]
# Audience: to a [demographic] | for [purpose]

# Example:
mdls "Explain machine learning using only sports metaphors for a business audience"
```
